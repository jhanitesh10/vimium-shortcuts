## Tab:
- t - Create a new tab
- yt- Duplicate current tab
- x - Close current tab
- X - Restore closed tab

## Scroll:
- h scroll left
- l scroll right
- h scroll left
- k scroll up
- j scroll down
- gg - scroll to top of the page
- G - scroll to the bottom of the page
- u - Scroll a half page up
- d - Scroll a half page down

## Navigation:
- H - Previous page
- L - Next page
- r - reload
- J - go one tab left
- K - go one tab right

## Search:
- / - Enter find mode
- n- Next
- N - prev match
